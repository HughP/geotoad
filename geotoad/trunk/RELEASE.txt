Release Instructions:
--------------------

GENERIC:
========
run tools/makedist.sh

WINDOWS:
========

You must create an EXE file with the exerb library:

ruby -r exerb/mkexr geotoad.rb
ruby C:\ruby\bin\exerb geotoad.exr

Then copy it into the CLI directory, and run tools/makedist.sh

Mac OS X:
=========
run tools/makedist.sh

Unzip the resulting Mac OS zip file, edit the icon for the .command file,
and make the file extension invisible for it. Then use Disk Utility to make
an image from the folder.
